# Changelog

Pacote com os componentes de UI para os clientes
[TabNews](https://tabnews.com.br) construídos em Flutter para
visualizar e interagir com as notícias e artigos publicados na plataforma.

## 0.1.0

### Theme

* Added theme engine
* Added dark and light colors
* Added theme extensions: colors, metrics and fonts

### Mixins

* Added `ThemeMixin`
* Added `DateTimeMixin`

### Widgets

* Added `CardWidget`
* Added `TextWidget`
* Added `IconWidget`
* Added `LoadingWidget`
* Added `GestureWidget`
* Added `NewsCardWidget`
* Added `WheelListViewWidget`
