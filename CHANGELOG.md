# Changelog

Package with UI components for [TabNews](https://tabnews.com.br) clients built
in [Flutter](https://flutter.dev) to view and interact with news and articles
published on the platform.

## 0.1.1

* Fix dependence path: `tabnewsnc_engine`

## 0.1.0

### Theme

* Added theme engine
* Added dark and light colors
* Added theme extensions: colors, metrics and fonts

### Mixins

* Added `ThemeMixin`
* Added `DateTimeMixin`

### Widgets

* Added `CardWidget`
* Added `TextWidget`
* Added `IconWidget`
* Added `LoadingWidget`
* Added `GestureWidget`
* Added `NewsCardWidget`
* Added `WheelListViewWidget`
